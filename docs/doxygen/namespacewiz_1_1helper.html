FieldEnabled ( field)

Checks if a field (or widget) is enabled.

This is used for compatibility between wx. 2.8 & 3.0. 3.0 uses the method 'IsThisEnabled()' rather than 'IsEnabled()' to get the 'intrinsic' status of the widget.

Parameters
fieldThe widget (wx.Window) to be checked

Detailed Description

Class defining a Debian package script.

A script's filename is one of 'preinst', 'prerm', 'postinst', or 'postrm'. Scripts are stored in the (FIXME: Don't remember section name) section of the package & are executed in the order dictated by the naming convention: 'Pre Install', 'Pre Remove/Uninstall', 'Post Install', & 'Post Remove/Uninstall'.

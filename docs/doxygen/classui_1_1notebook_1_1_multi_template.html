Detailed Description

Multiple instances of a single template.

A ui.layout.BoxSizer that creates a ui.notebook.Notebook instance for creating multiple templates on a single wiz.wizard.WizardPage.

Parameters
parentThe wx.Window parent instance
panelClasswx.Panel derived class to use for tab pages

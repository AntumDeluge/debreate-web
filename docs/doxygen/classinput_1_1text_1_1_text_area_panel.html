TextAreaPanel.Enable ( args, kwargs )

Disables or enables self & text area.

Because of issues with text control background color, it must be set manually when the parent TextAreaPanel is disabled. This causes some text artifacts in wx 3.0 when using the SetValue & WriteText methods. Because of that, when the control is enabled/disabled, the value is stored, then the control cleared. When the new background color is set, the value is restored, causing the text to take on the same background color.

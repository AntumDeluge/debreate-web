Member Function Documentation

ProgressDialog.Cancel ( )

Sets the active status to False.

Calls ui.progress.ProgressDialog.OnAbort

ProgressDialog.Destroy ( args, kwargs )

Closes & destroys the dialog.

Method for compatibility between wx 3.0 & older versions For 3.0 & newer, simply overrides wx.ProgressDialog.Destroy For older versions, calls ui.progress.ProgressDialog.EndModal

ProgressDialog.GetGauge ( )

Retrieves the wx.Gauge child object.

Mostly for compatibility with wx versions older than 3.0 Older versions do not have many of methods for access & manipulation of the gauge

ProgressDialog.GetMessage ( args, kwargs )

Retrieves the message shown on the dialog.

Method for compatibility between wx 3.0 & older versions For wx 3.0 & newer, simply overrides wx.ProgressDialog.GetMessage For older versions, finds the child wx.StaticText object & returns the label

ProgressDialog.GetRange ( args, kwargs )

Retrieves the range, or maximum value of the gauge.

Method for compatibility between wx 3.0 & older versions For 3.0 & newer, simply overrides wx.ProgressDialog.GetRange For older versions, calls GetRange on the child gauge object

ProgressDialog.GetValue ( args, kwargs )

Retrieves the current value of the gauge.

Method for compatibility between wx 3.0 & older versions for 3.0 & newer, simply overrides wx.ProgressDialog.GetValue For older versions, calls GetValue() on the child wx.Gauge object

ProgressDialog.OnAbort ( event=None )

Sets the active status to False.

The dialog will be destroyed when ui.progress.ProgressDialog.WasCancelled is called
